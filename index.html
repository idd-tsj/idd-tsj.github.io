Setting Up A Project

  1. Create a project folder

    • title project folder
    • create css, ig, js, and planning folders inside the project folder
    • create index.css and reset.css files inside css folder
    • create index.html file in project folder
    • create .gitignore and README.md filesin project folder

  2. Set up index.html file

    • designate the file as a html document --> "!DOCTYPE HTML"
    • create the "head" section with a "title"
    • link your stylesheet to your html document--> "link rel="stylesheet" href="css/index.css"" "
    • create a "body"

  3. Initialize Project Respository in Git Hub

    • select the git icon in the Visual Studio Explorer and choose project file
    • select all files that you wish to track
    • create the very first version and message titled 'first commit'
